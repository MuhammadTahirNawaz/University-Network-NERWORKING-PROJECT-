# University Network Project

This project designs and implements a comprehensive **University Network** infrastructure. It utilizes advanced networking concepts, including **Variable Length Subnet Masking (VLSM)** and **Virtual Local Area Networks (VLANs)**, to manage traffic and security across different campus departments.

---

### Network Overview

The system architecture is divided into six distinct functional units:
* **CS** (Computer Science)
* **CE** (Computer Engineering)
* **ME** (Mechanical Engineering)
* **CERAD**
* **ADMIN**
* **BE**

---

### Key Features

* **Centralized Services**: Includes a dedicated **Web Server** and **Email Server** hosted within the network.
* **Inter-Unit Communication**: All units are configured to send and receive emails from one unit to another.
* **Security & Isolation (CERAD)**: The **CERAD** unit is restricted; it can merely make links within the unit and is unable to access other units.
* **Functional Connectivity**: The design provides individual network access and functional links between units.

---

### Project Structure

* **Concepts Used**: Variable Length Subnet Masking (VLSM) and Virtual Local Area Networks (VLANs).
* **Topology**: A multi-unit router-based configuration connecting departmental subnets via switches and access points.
